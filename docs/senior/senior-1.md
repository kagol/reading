# 必修1

## 第一单元

这个单元主要学习现代新诗。

新诗是五四前后才出现的。

读诗可以陶冶性情，可以学习用精炼的语言和新颖的意象来表达情意。鉴赏诗歌应在反复诵读的基础上，着重分析意象，同时品味语言，发挥想象，感受充溢于作品的真情。

- 1 沁园春·长沙 / 毛泽东
- 2 诗两首
  - 雨巷 / 戴望舒
  - 再别康桥 / 徐志摩
- 3 大堰河——我的保姆 / 艾青
